Libraries in Python

Numpy NumPy is the fundamental package for scientific computing in Python.Numpy

Pandas Pandas is an open source, BSD-licensed library providing high-performance, easy-to-use data structures and data analysis tools for the Python programming language.Pandas

Matplotlib Matplotlib is a comprehensive library for creating static, animated, and interactive visualizations in Python.

Seaborn This is also another visualization tool, but more focused on statistical visualization.

Scikit-Learn Simple and efficient tools for predictive data analysis. One stop shop for all Machine learning algorithms written in Python SKLearn

Flask Python Framework for Machine Learning API Creation.

Data Analysis With help of all mentioned tools above we are trying to develop a Machine learning model , we analysis the data , train data , test data and predict the outcome .

Platform for Machine Learning You can use Jupyter Notebooks or your favorite choice of editor , Jupyter notebooks have all the pre installed libraries that you need for ML. Or you can manually install them as you need.

Data Analysis

  • Clean the data
  • Remove Duplicates in Data
  • Remove Nulls in the data
  • Identify the Categorical Data and numerical data
  • Plot the correlation or heat maps
  • Feature Engineering

Model Selection in Machine learning Models are classified in to two broad categories

  • supervised learning – Supervised learning (SL) is the machine learning task of learning a function that maps an input to an output based on example input-output pairs

  • unsupervised learning – Unsupervised learning algorithms take a set of data that contains only inputs, and find structure in the data, like grouping or clustering of data points

Widely Commonly used Algorithms are :

  • Linear regression
  • Logistic regression
  • Decision trees
  • K-nearest neighbor algorithm
  • and lot more

Data is trained using sklearn library and once the data is trained , we use any of above mentioned algorithm based on our business purpose. Once the Model is trained we evaluate our model and evaluation of Machine Learning model is a core part of building an effective machine learning model. Different kind of evaluation are used for different purposes . Basic Evaluations are Confusion Matrix ,F1 Score and Root Mean Squared Error.

Once the Model is perfect and the performance is good , we need to deploy the application using frame works like Flask . As a software professional , we use many CI/CD pipeline to deploy our mode as well.
